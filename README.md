# ChatVerse

A full-stack real-time chat application built with FastAPI, React, PostgreSQL, JWT Authentication, and WebSockets.

## Features

- User Signup/Login
- JWT Authentication
- Real-time Messaging
- Persistent Chat History
- Create, Rename, Delete Rooms
- Search Rooms
- Responsive UI

## Tech Stack

Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- WebSockets
- JWT

Frontend
- React
- Axios
- React Router

Database
- PostgreSQL

## Installation

### Backend

pip install -r requirements.txt

uvicorn app.main:app --reload

### Frontend

npm install

npm run dev
